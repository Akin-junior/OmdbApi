# Omdb Search Api:

Search movie and series with omdbApi.


## Key features

 * ✔ Mvvm
 * ✔ BaseFragment
 * ✔ Kotlin Ext
 * ✔ Retrofit
 * ✔ Navigation Component
 * ✔ View-Binding
 * ✔ Motion Layout
 * ✔ Glide
 * ✔ Lottie Animation




## Introduction
* We can search movies and series using omdb search api.
* It is possible to go to the detail page and see some details.
